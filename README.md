# Epic game

- Epic game is a card-based nft game powered by [@buildspace] (https://app.buildspace.so/)
- This project is built with `solidity` `hardhat` `react` `javascript`

## How to play

```
   cd web
   `yarn` to install dependencies
   `yarn start`

```
